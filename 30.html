About Ruby on Rails
Ruby on Rails is an open source, MVC (Model-View-Controller)
framework used to build websites and applications.
Rails combines the Ruby programming language with HTML,
CSS, and JavaScript to create a web application that runs on a
web server. Because it runs on a web server, Rails is considered
a server-side, or “back end,” web application development
platform (the web browser is the “front end”).
