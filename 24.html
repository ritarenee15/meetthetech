Other loops
While loops: run until loop
evaluates to false
while x >= 0
puts x
x = x - 1
end
puts "Done!"
Until loops: opposite of while
loops
until x < 0
puts x
x -= 1
end
puts "Done!"
Conditionals within loops:
additional control flow
while x <= 10
if x.odd?
puts x
end
x += 1
end
